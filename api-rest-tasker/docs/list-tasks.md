# List all tasks

Show all tasks available

**URL** : `/tasks/`

**Method** : `GET`

## Success Responses

**Code** : `200 OK`

**Content** : In this example, the User have one task:

```json
[
    {
        "id": 1
        "description": "first task"
        "due_date": "2022-05-01 12:00"
    }
]
```
